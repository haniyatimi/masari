# Masari V2 — المرحلة الأولى

نسخة ثابتة تعمل مباشرة على GitHub Pages.

## الملفات

- `index.html`: تسجيل الدخول وإنشاء الحساب.
- `dashboard.html`: لوحة التحكم والمضمار.
- `css/style.css`: جميع التنسيقات.
- `js/auth.js`: الحساب المحلي.
- `js/dashboard.js`: العادات والمضمار والتحليل.
- `assets/logo.svg`: شعار Masari.

## التشغيل

افتح `index.html` في المتصفح، أو ارفع جميع الملفات والمجلدات إلى GitHub.

## النشر على GitHub Pages

1. ارفع **محتويات** مجلد `masari-v2` إلى جذر مستودع `masari`.
2. افتح `Settings`.
3. اختر `Pages`.
4. اختر `Deploy from a branch`.
5. اختر الفرع `main` والمجلد `/(root)`.
6. اضغط `Save`.

## ملاحظة

الحسابات والبيانات تُحفظ محليًا في المتصفح باستخدام `localStorage`.
Firebase سيُضاف لاحقًا بعد اعتماد التصميم.
